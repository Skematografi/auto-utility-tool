# Auto Utility Tool

A lightweight utility toolkit designed to simplify common tasks and improve productivity.

## Features

* Auto Calculator by List Data
* Find Duplicate by List Data
* Detect Non-ASCII Characters

## Run Locally

Clone this repository:

```bash
git clone https://github.com/Skematografi/auto-utility-tool.git
```

Open the project folder and run:

```text
index.html
```

Simply open `index.html` in your browser — no installation or setup required.
